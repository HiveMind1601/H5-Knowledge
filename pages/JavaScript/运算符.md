关系运算符(结果为布尔值)
<,>,<=,>=,==,===,!=(不相等),!==(不全等)
<= 中 < 与 = 是或的关系
= 是赋值符号
== 比较值是否相等,具有隐式转换(数字为底层)
=== 恒等(数据类型和值都要相等)
alert("5"=5);//true
alert("5"===5);//false:类型不相等
